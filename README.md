# SmartTask Orchestrator (FWD Edition)

A responsive, client-side task management dashboard built to visualize workload urgency and optimize personal productivity.

##  Key Features
* **Deadline-Driven UI:** Color-coded task cards (Critical, Planned, Ongoing).
* **Dynamic Theme Engine:** Seamless Light/Dark mode toggling using CSS Variables.
* **Real-time Analytics:** Automated progress tracking and total workload calculation.
* **Responsive Design:** Optimized for mobile, tablet, and desktop views.

##  Tech Stack
* **HTML5:** Semantic structure for task organization.
* **CSS3:** Custom properties (Variables), Flexbox, and Grid.
* **JavaScript:** Vanilla JS for state management and DOM manipulation.

##  Project Structure
* `index.html` - Application structure.
* `style.css` - Theming and layout logic.
* `script.js` - Interactive functionality and data handling.
